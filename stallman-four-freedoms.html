This page contains contains a video using a video element and controls provided by the media player plyr. The video element contains two track elements: one for an MP4 version of the video and one for an Ogg Vorbis version of the video. The video element also contains several track elements for various caption and subtitle formats: WebVTT, SBV and SRT. For a version of this page that uses the browser's native controls for the video element, see the page Test Video for Native Video Player Controls.
